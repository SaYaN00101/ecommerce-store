## ğŸš€ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SaYaN00101/ecommerce-store.git
   cd ecommerce-store
   ```

2. **Set up the database**
   - Import the database schema from `database/mystore.sql`
   - Configure database connection in `includes/connect.php`

3. **Configure web server**
   - Set up Apache/XAMPP/WAMP
   - Point document root to the project directory
   - Ensure PHP version 7.4+ is installed

4. **Start the application**
   - Access through localhost in your web browser
   - Default admin credentials will be provided separately
